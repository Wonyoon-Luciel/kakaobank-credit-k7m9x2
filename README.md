# 카카오뱅크 신용정보 프로토타입

iPhone 17 Pro Max 최적화 웹 애플리케이션

## 기능
- 6개 화면 구현 (홈, 신용정보, 카드, 대출, 연체, 보증)
- URL 기반 라우팅
- 카카오뱅크 UI/UX 재현

## 사용 방법
`index.html` 파일을 브라우저에서 열거나 GitHub Pages로 배포

## 화면
- 홈: `index.html`
- 신용정보: `index.html#credit`
- 카드: `index.html#card`
- 대출: `index.html#loan`
- 연체: `index.html#overdue`
- 보증: `index.html#guarantee`
